# CrowdSense AI 🚀

A real-time intelligent crowd management system designed for large-scale sporting venues.

## 🧠 Problem
Large stadiums suffer from:
- overcrowding
- long waiting times
- poor navigation
- lack of real-time insights

## 💡 Solution
CrowdSense AI provides:
- Live crowd density heatmap
- AI-powered smart suggestions
- Optimal route planning avoiding congestion
- Real-time alert system
- Admin dashboard for operational insights

## ⚡ Key Features
- Dynamic crowd simulation engine
- Intelligent routing using congestion-aware logic
- Live AI recommendations ("Best Action Right Now")
- Predictive congestion analytics
- Smooth, production-grade UI/UX

## 🧩 Tech Stack
- Vanilla JavaScript (ES Modules)
- Vite
- HTML/CSS (custom design system)
- Chart.js

## 🎯 What Makes It Special
Unlike traditional dashboards, CrowdSense AI:
- Doesn’t just show data
- It actively guides users in real-time
